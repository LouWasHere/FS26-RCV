# FS26 Leeds Gryphons ReCeiVer "Rocky"

This is the repository for the Leeds Gryphon Racing FS26 DAQ Receiver system. It contains primarily code for flashing onto a Pico to receive data over radio, but also contains the web-app interface I built to visualize, monitor and store received data more easily.

For further information, please refer to the Wiki. If you're the unlucky soul who's been given the job of maintaining this part of the FS project, welcome! Please get in touch with me so that I can give you access to this repo, or fork your own copy if you so desire.
